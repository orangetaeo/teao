주문 기본설정

주문상태 설정
입금대기 주문 초기상태, 무통장과 가상계좌가 해당됩니다.
입금상태 상태 설정
예약금완료 예약금 확인된 상태입니다.. 잔금대기 예약금 완료 상태에서, 잔금입금 대기상태입니다.
결제완료 입금확인된 상태, 또는 결제완료된 상태입니다.
확정/미확정 예약관리
여행상품 미확정 랜드사에 여행상품을 확인 하는 단계입니다. 여행상품 확정 여행상품이 확정 된 단계입니다.
예약접수 소비자가 여행상품을 예약접수한 상태입니다. 예약가능 관리자가 자동으로 예약가능 상태로 만듭니다.
예약확정 관리자가 예약확정하는 단계입니다. 예약완료 관리자가 예약완료하는 단계입니다.
구매 상태 설정
구매확정 구매확정된 상태입니다.(고객이 구매확정하거나, 관리자가 상태변경)
취소 상태 설정 일 동안 미입금시 주문자동취소 자동취소를 원하지 않는 경우 0으로 설정
취소 설정
자동취소 주문접수 후 오랜동안 미입금 되거나, 가상계좌 만료된 상태입니다. 자동취소완료

  • '결재대기'상태에서 주문접수 후 일 후 '자동취소'로 자동 주문상태 변경

  • 사용안함
품절취소 주문접수 후 여행상품 재고가 없어 취소된 상태입니다.
관리자취소 주문접수 후 관리자가 여러 원인에의해 임의 취소한 상태입니다. 고객취소요청 주문접수 단계에서 고객이 취소요청을 한 상태입니다.
실패 상태 설정
고객결제중단 고객이 결제완료 전 PG사 결제창을 닫거나 다른페이지로 이동한 상태입니다. 결제실패 고객결제 후 PG사에서 결제실패 결과값을 받은 상태입니다.
PG확인요망 고객결제 후 PG사에서 결과값을 받지 못하여 PG사에서 확인이 필요한 상태입니다.
환불상태 설정
환불접수 입금확인 또는 반품요청 이후 관리자가 환불 처리하는 단계입니다. 환불보류 고객이 접수한 환불요청을 보류처리한 상태입니다.
환불완료 환불이 완료된 상태로, 해당주문상품이 취소완료된 상태입니다.
